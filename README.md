# Stock-Ticker-Vuejs

## Project Setup

### Clone the project
```sh
git clone https://github.com/Doaa-Ismael/Stock-ticker-app.git
```
### Install dependencies
```sh
yarn install
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Type-Check, Compile and Minify for Production

```sh
yarn build
```

### Lint with [ESLint](https://eslint.org/)

```sh
yarn lint
```
